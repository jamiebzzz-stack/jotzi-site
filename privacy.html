Jotzi Privacy Policy

Effective date: July 7, 2026

Publisher: Ocoee Studios LLC

Overview

Jotzi is designed as a gentle reminder companion. The app may let you create reminders, Smart Loops, sparks, private Vault notes, recipes, and settings.

Data currently handled by the prototype

The current Expo prototype is designed around local app interactions. User-entered reminders, sparks, recipes, Vault items, settings, and preview health values are not currently connected to a production server in this prototype.

HealthKit and Apple Watch

Jotzi contains product concepts for future Apple Watch and HealthKit features. If those features are implemented, Jotzi will request permission before accessing HealthKit data. Health information should only be used to support reminder timing and user experience.

Face ID and Vault

Jotzi contains a Vault concept for protecting private reminders and notes. If native biometric unlock is implemented, Face ID authentication will be handled through device-level Apple APIs.

No sale of personal data

Ocoee Studios LLC does not sell Jotzi user data.

Contact

For privacy questions, contact Ocoee Studios LLC at: support@ocoeestudios.com
